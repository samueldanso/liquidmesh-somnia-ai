### User Flow

1. Connect wallet.
2. Entry guard routes: active position → Dashboard; else → Deposit flow.
3. Deposit stepper:
   - Faucet: wrap STT → wSTT and mint USDC
   - Deposit: approve and deposit to LiquidityVault
   - Activate: enable automation (interval/cooldown)
4. Dashboard: Agent Status, Positions, Pool metrics, Activity
